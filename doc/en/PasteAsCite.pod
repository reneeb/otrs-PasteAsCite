=head1 NAME

PasteAsCite - Every copypasted text in the richtext editor will be handled as a cite.

=head1 DESCRIPTION

This addon allows you to copy text from the "cited" text in the answer dialog (and others, too)
and when you paste the text, it's still a cite.
